Study the following concepts, Prepare some examples to demonstrate these concepts:
		a.  Static Class
		b.  Partial Class
		c.   Sealed Class 

a. Static Class
 Key Points:
Cannot be instantiated.

Can contain only static members (methods, properties, etc.).

Useful for utility/helper functions.

Example:

using System;

static class MathHelper
{
    public static int Add(int a, int b)
    {
        return a + b;
    }

    public static int Square(int x)
    {
        return x * x;
    }
}

class Program
{
    static void Main()
    {
        Console.WriteLine("Sum: " + MathHelper.Add(5, 3));
        Console.WriteLine("Square: " + MathHelper.Square(4));
    }
}

b. Partial Class
Key Points:
Allows splitting a class across multiple files or parts.

Useful for teamwork or separating autogenerated and manual code.


// Part 1
public partial class Person
{
    public string FirstName;
    public string LastName;
}

// Part 2
public partial class Person
{
    public void Display()
    {
        Console.WriteLine($"Name: {FirstName} {LastName}");
    }
}

class Program
{
    static void Main()
    {
        Person p = new Person();
        p.FirstName = "John";
        p.LastName = "Doe";
        p.Display();
    }
}


 c. Sealed Class
 Key Points:
A sealed class cannot be inherited.

using System;

// Sealed class
sealed class Vehicle
{
    public void Drive()
    {
        Console.WriteLine("Driving the vehicle.");
    }
}

// ‚ùå This would cause a compile-time error
// class Car : Vehicle { }

class Program
{
    static void Main()
    {
        Vehicle v = new Vehicle();
        v.Drive();
    }
}

Used when you want to prevent further extension of a class.
